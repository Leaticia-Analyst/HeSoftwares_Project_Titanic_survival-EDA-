# Titanic Survival Prediction(EDA)

## Project Overview
This project analyzes the Titanic passenger dataset and builds machine learning models to predict whether a passenger survived or not.

The project demonstrates the full machine learning workflow including data preprocessing, feature engineering, model training, and evaluation.

The dataset contains passenger information such as:
- Passenger class
- Age
- Sex
- Fare
- Family relations
- Embarked port

## Project Steps
1. Data Understanding
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Model Training
6. Model Evaluation

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors
- Support Vector Machine

## Evaluation Metrics
The models were evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Results
Among all tested models, **Random Forest achieved the best performance** with the highest accuracy.
